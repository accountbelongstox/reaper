接口说明

所有数据

请求地址:http://api.gongying.love/api

请求类型：GET

请求参数：datatypes
支持类型：shop_score [返回：店铺评分数据]
支持类型：smart_business_opportunity [返回：智慧商机数据]
可以两者并用,如:shop_score,smart_business_opportunity

请求参数：username
需要请求的用户名,如:zsw100023649

示例请求：

http://api.gongying.love/api?datatypes=shop_score,smart_business_opportunity&username=zsw100023649

javascript代码：
let data = {
  datatypes:'shop_score,smart_business_opportunity',
  username:'zsw100023649'
}
$.get('http://api.gongying.love/api',data,function(data){
  if(data.message == 'successfully get data'){
    console.log(data.data)
  }
})
